# AmberStudent Clone (React.js)

##  Project Overview

This project is a frontend clone of the AmberStudent website built using React.js.
It replicates the core UI and basic functionalities such as authentication, property listings, and search.

---

##  Features

* User Authentication (Login / Signup using LocalStorage)
* Dynamic Property Listings
* Search functionality (filter by city)
* Protected Routes
* Responsive Design
* Modern UI with reusable components

---

## 🛠️ Tech Stack

* React.js
* React Router DOM
* Context API
* CSS (Flexbox & Grid)

---

## 📂 Project Structure

src/
├── components/
├── pages/
├── context/
├── data/
├── App.js
└── index.js

---

## Authentication Details

* Users can sign up and log in
* Data is stored in LocalStorage
* Protected routes restrict access to certain pages

---

## Learning Outcomes

* Understanding of React component-based architecture
* Implementation of authentication using Context API
* Handling state and routing in React applications
* Building responsive and modern UI
